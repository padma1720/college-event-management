# Todo List Web Application

This is a simple Todo List web application built using Django.

## Features

- User Registration
- User Login
- Add Tasks
- Edit Tasks
- Delete Tasks
- View Task List

## Technologies Used

- Python
- Django
- HTML
- CSS
- SQLite

## Installation

1. Clone the repository

git clone https://github.com/padma1720/todolist.git

2. Install requirements

pip install -r requirements.txt

3. Run server

python manage.py runserver
